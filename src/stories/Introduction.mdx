# Introduction

<br />
## Welcome!

- **Basic Design System**은 Storybook을 이용하여 UI Components의 가이드라인을 제시합니다.
- 색상, 폰트부터 UI Components까지, 프로젝트의 디자인에 일관성을 부여하기 위해 제작되었습니다.
- UI Components의 모양을 살펴보고 직접 arguments를 변경하여 UI의 동작을 확인해보세요.

<br />
## How to use

<br />

#### 1. UI Components 둘러보기

- 다양한 UI Components를 확인해보세요
- Docs를 통해 해당 Component의 전체적인 요약을 볼 수 있으며, 각 Component의 목록을 클릭하면 자세한 내용을 확인할 수 있습니다.

#### 2. Arguments 변경

- Docs의 Control에서 Arguments를 변경할 수 있습니다.
- 직접 내용을 변경하여 Component의 동작을 확인해보세요.
